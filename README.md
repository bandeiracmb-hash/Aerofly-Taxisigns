# Aerofly-Taxisigns
Taxisigns for several airports in Aerofly FS4 - PC version

Installation:
Folders are named with the following format:coordinates_icao_taxisigns
For example e00843n4537_limc_taxisigns for LIMC

Put these folder inside your \$User\Documents\Aerofly FS 4\scenery\poi

Airports so far:
LEIB, LEMH, LFML, ELLX, LFKB, LIPX, LOWI, LOWS, LEAL
